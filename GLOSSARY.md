# Glosarium

- **Frontend statis**: situs yang disajikan langsung dari file HTML/CSS/JS tanpa render server-side.
- **Backend API**: layanan server yang menyediakan data dan proses bisnis melalui endpoint HTTP.
- **Endpoint**: URL API untuk operasi tertentu (contoh: `/auth/login`, `/orders`).
- **Snap (Midtrans)**: mekanisme pembayaran Midtrans yang memunculkan UI pembayaran di sisi client.
- **Cart**: keranjang belanja; dapat disimpan lokal maupun di server.
- **CORS**: kebijakan browser yang mengatur akses lintas-domain.
- **Webhook**: callback dari payment gateway ke backend saat status transaksi berubah.
- **Static hosting**: penyajian file statis oleh web server tanpa build runtime.
